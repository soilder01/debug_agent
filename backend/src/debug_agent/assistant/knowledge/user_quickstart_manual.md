# 小D用户快速开始和完整使用手册

## 文档定位
- 本文档主题：用户操作手册。
- 主要读者：第一次使用小D的评测、算法和业务同学。
- 文档用途：用于小D问答、项目验收、用户培训、运行排障和后续知识沉淀。
- 文档要求：每一条规则都应能被追溯到用户体验、工程状态、证据链或审计记录。

- 摘要：本文档面向真实用户，说明如何提交badcase、如何处理表格、如何看进度和报告。
- 摘要：用户不需要理解后端API，只需要按飞书卡片和用户视图完成闭环。
- 摘要：本文档也是首次使用卡片的完整手册链接目标。
